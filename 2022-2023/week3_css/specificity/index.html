Specificity test

How to calculate specificity

  • Inline styling: 1000
  • IDs: 100
  • Classes and pseudo classes: 10
  • Elements and pseudo elements: 1

Find specificity

  • h1.example
  • h1#example
  • #example
  • <h1 style="color: red;"> </h1>
  • #main p#hidden

More rules

  1. If both specificity values are the same, the latest one wins
  2. Internal stylesheet wins over external stylesheet
  3. The universal selector * has specificity value 0
